    类 AnimationManager

    模型动画管理器，提供对多个AnimationClip的管理，包括获取，播放，暂停，切换等功能

    playTime

    • get playTime(): number

      获取当前播放时间

      返回 number

      当前播放时间（秒），如果没有动作则返回0

    • set playTime(time: number): void

      设置当前播放时间

      参数

      • time: number

        要设置的时间（秒），会自动限制在 [0, duration] 范围内

      返回 void

    reverse

    • get reverse(): boolean

      获取是否倒放

      返回 boolean

    • set reverse(value: boolean): void

      设置是否倒放

      参数

      • value: boolean

        是否倒放，默认为false

      返回 void

    speed

    • get speed(): number

      获取播放速度

      返回 number

    • set speed(value: number): void

      设置播放速度

      参数

      • value: number

        播放速度，默认为1，大于1表示加速，小于1表示减速

      返回 void
