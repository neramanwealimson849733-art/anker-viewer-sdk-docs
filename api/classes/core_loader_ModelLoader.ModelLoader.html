    类 ModelLoader

    模型加载器 职责：

    1. 从文件/URL 加载模型
    2. 使用 ResourceManager 管理缓存
    3. 协调场景添加
    4. 协调相机自适应
    5. 业务逻辑编排

    constructor

    • new ModelLoader(
          dracoPath: string,
          resourceManager: ResourceManagerClass,
          cameraManager?: CameraManager,
      ): ModelLoader

      构造函数

      参数

      • dracoPath: string

        Draco 解码器路径

      • resourceManager: ResourceManagerClass

        资源管理器（用于缓存）

      • 可选cameraManager: CameraManager

        相机管理器（可选，用于自动适配）

      返回 ModelLoader

    方法

    load

    • load(params: ModelParams): ModelObject | undefined

      【同步方法】加载模型（回调方式）

      参数

      • params: ModelParams

        模型参数

      返回 ModelObject | undefined

      立即返回 ModelObject（可能是占位对象）或 undefined

      示例

      const loader = new ModelLoader(dracoPath, resourceManager, cameraManager);
      const model = loader.load({
        urlOrId: '/path/to/model.glb',
        onLoad: (model) => {
          console.log('Model loaded:', model);
        }
      });

    loadAsync

    • loadAsync(params: ModelParams): Promise<ModelObject>

      【异步方法】加载模型（Promise 方式）

      参数

      • params: ModelParams

        模型参数选项

      返回 Promise<ModelObject>

      Promise

      示例

      try {
        const loader = new ModelLoader(dracoPath, resourceManager, cameraManager);
        const model = await loader.loadAsync({
          urlOrId: '/path/to/model.glb'
        });
        console.log('Model loaded:', model);
      } catch (error) {
        console.error('Failed to load:', error);
      }
