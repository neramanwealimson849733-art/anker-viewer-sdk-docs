    类 BottomShadow内部成员

    底部阴影类 参考 Google model-viewer 的 Shadow.ts 实现

    核心特点：

    1. 使用归一化的正交相机视锥体（-0.5 到 0.5）
    2. 通过 camera.scale 调整实际覆盖范围
    3. 自动检测动画模型并采样所有关键帧计算最大包围盒
    4. 阴影平面基于整个动画活动范围，防止裁切
    5. 模糊边距通过 camera.scale 补偿，避免裁切
