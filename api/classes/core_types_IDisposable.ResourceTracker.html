    类 ResourceTracker

    资源追踪器（开发模式使用）

    用于跟踪资源的创建和释放，帮助发现内存泄漏。

    示例

    const tracker = new ResourceTracker();
    const viewer = tracker.track(new Viewer(container));
    const model = tracker.track(await viewer.loadModelAsync({...}));

    // 使用完毕后释放
    tracker.dispose();

    // 检查是否有未释放的资源
    console.log('Active resources:', tracker.getActiveCount());
