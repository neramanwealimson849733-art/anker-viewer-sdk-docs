    dispose

    • dispose(): void

      释放查看器及所有资源

      调用此方法后，查看器将不可再使用。 释放的资源包括：

      • 渲染器（WebGLRenderer）
      • 所有场景和模型
      • 相机控制器
      • 所有组件
      • 事件监听器
      • 资源缓存

      返回 void

      示例

      const viewer = new Viewer(container);
      // ... 使用 viewer
      viewer.dispose(); // 释放资源

      实现了 IDisposable.dispose

        模型参数

      返回 Promise<ModelObject>

      Promise

      示例

      try {
        const model = await viewer.loadModelAsync({
          urlOrId: '/models/product.glb'
        });
        console.log('Model loaded:', model);
      } catch (error) {
        console.error('Failed to load:', error);
      }
