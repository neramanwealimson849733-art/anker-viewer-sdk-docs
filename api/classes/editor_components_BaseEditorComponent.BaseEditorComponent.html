    类 BaseEditorComponent抽象

    编辑器组件基类

    提供模型编辑器和场景编辑器的共享功能，包括：

    • 状态管理（通过 EditorState）
    • 材质管理（通过 MaterialManager）
    • 环境管理（环境贴图、强度、旋转）
    • 背景设置（纯色、纹理、使用环境贴图）
    • 渲染参数（色调映射、曝光度）
    • 属性编辑（通过 PropertyEditor）
    • 对象选择（通过 SelectionManager）

    子类需要实现：

    • serialize(): 序列化编辑数据
    • load(): 加载编辑数据

      参数

      • 可选options: EnvironmentLoadOptions

        加载选项（可选）

      返回 Promise<void>

      示例

    setToneMapping

    • setToneMapping(type: number): void

      设置色调映射

      参数

      • type: number

        色调映射类型（Three.js 常量）

        可用类型：

        • THREE.NoToneMapping (0)
        • THREE.LinearToneMapping (1)
        • THREE.ReinhardToneMapping (2)
        • THREE.CineonToneMapping (3)
        • THREE.ACESFilmicToneMapping (4)

      返回 void

      示例

      import { ACESFilmicToneMapping } from 'three';
      editor.setToneMapping(ACESFilmicToneMapping);
