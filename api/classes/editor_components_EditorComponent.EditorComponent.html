    类 EditorComponent

    编辑器组件 提供场景编辑器的核心功能，包括对象选择、变换、属性编辑等

    示例

    const viewer = new Viewer(container);
    const editor = new EditorComponent();
    viewer.addComponent(editor);

    // 设置编辑模式
    editor.setMode('translate');

    // 监听选择事件
    editor.addEventListener('objectSelected', (e) => {
      console.log('Selected:', e.object);
    });
