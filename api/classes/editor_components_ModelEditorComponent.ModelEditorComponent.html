    类 ModelEditorComponent

    模型编辑器组件

    提供单个模型的编辑功能，包括：

    • SKU 管理（创建、切换、删除）
    • 材质配置编辑
    • 环境和背景设置
    • 渲染参数设置
    • 相机视角管理
    • 模型配置的序列化和加载

    使用场景：

    • 编辑单个 3D 模型的显示效果
    • 创建和管理多个 SKU（款式）
    • 保存模型的默认展示配置

    createSKU

    • createSKU(
          name: string,
          description?: string,
          _captureEnvironment?: boolean,
      ): SKU

      创建新的 SKU

      捕获当前模型的材质配置、环境设置等，创建一个新的 SKU。 新创建的 SKU 会被自动添加到模型的 SKU 列表中，并设置为当前 SKU。

      参数

      • name: string

        SKU 名称

      • 可选description: string

        SKU 描述（可选）

      • _captureEnvironment: boolean = false

        是否捕获环境设置（默认 false）

      返回 SKU

      创建的 SKU 对象

      示例

      // 创建基础 SKU（仅捕获材质）
      const blackSKU = modelEditor.createSKU('黑色款', '经典黑色配色');

      // 创建完整 SKU（包含环境、渲染参数等）
      const premiumSKU = modelEditor.createSKU('高级款', '高级渲染效果', true);

      参数

        SKU 的唯一标识

      返回 void

      示例

      modelEditor.deleteSKU('sku_black_001');
