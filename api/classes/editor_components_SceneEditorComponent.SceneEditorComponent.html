    类 SceneEditorComponent

    场景编辑器组件

    提供场景编辑功能，包括：

    • 场景对象管理（添加、删除、组合）
    • 层级管理（父子关系、可见性、锁定）
    • 变换控制（移动、旋转、缩放）
    • 场景序列化和加载

    使用场景：

    • 组合多个模型创建场景
    • 管理场景中的对象层级
    • 调整对象的位置、旋转、缩放
    • 保存和加载完整场景配置

    重要限制：

    • 场景编辑器不直接编辑模型材质
    • 场景对模型是引用关系，不影响原始模型
    • 材质变化通过切换SKU实现

    addObject

    • addObject(
          model: ModelObject,
          options?: Omit<AddObjectOptions, "skuId">,
      ): ModelObject

      添加对象到场景

      参数

      • model: ModelObject

        要添加的模型对象

      • 可选options: Omit<AddObjectOptions, "skuId">

        添加选项（可选）

      返回 ModelObject

      添加的模型对象

      示例

      const model = await viewer.loadModel({ urlOrId: 'M3EF684585' });
      sceneEditor.addObject(model, {
        position: [0, 0, 0],
        rotation: [0, Math.PI / 4, 0],
        scale: [1, 1, 1],
        name: '产品A',
        autoSelect: true
      });
