    类 MaterialManager

    MaterialManager - 材质管理器

    职责：

    1. 整合 MaterialQueryService 和 MaterialEditService
    2. 管理材质选择状态
    3. 协调状态更新和事件分发
    4. 提供统一的材质管理接口

    架构位置：Manager 层

    • 依赖：QueryService, EditService, EditorState
    • 被依赖：UI Components, EditorViewer
