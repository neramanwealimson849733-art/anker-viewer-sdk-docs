    类 SKUManager

    SKUManager - SKU 管理器

    职责：

    1. 管理 SKU 数据的加载和缓存
    2. 处理 SKU 切换逻辑
    3. 协调材质替换
    4. 管理 SKU 状态
    5. 触发 SKU 相关事件

    架构位置：Manager 层

    • 依赖：MaterialManager, EditorState
    • 被依赖：UI Components, EditorViewer
