    类 MaterialEditService

    材质编辑服务 提供材质属性的修改功能，支持批量操作

    示例

    const editService = new MaterialEditService();

    // 监听属性变化
    editService.addEventListener('propertyChanged', (e) => {
      console.log(`${e.property} changed`);
    });

    // 编辑材质属性
    editService.setMaterialColor(material, '#ff0000');
    editService.setMaterialMetalness(material, 0.8);

    // 批量编辑
    editService.beginBatch();
    editService.setMaterialColor(material, '#ff0000');
    editService.setMaterialMetalness(material, 0.8);
    editService.setMaterialRoughness(material, 0.2);
    editService.endBatch(); // 只触发一次事件
