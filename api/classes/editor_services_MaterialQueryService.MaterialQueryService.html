    类 MaterialQueryService

    材质查询服务 提供材质的只读查询功能，包括缓存优化

    示例

    const queryService = new MaterialQueryService();
    queryService.setTarget(modelObject);

    // 获取所有材质（已去重和缓存）
    const materials = queryService.getMaterials();

    // 根据材质查找对象
    const object = queryService.findObjectByMaterial(materials[0]);
