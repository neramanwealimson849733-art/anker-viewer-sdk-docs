    类 EditorState

    编辑器状态管理 作为单一数据源（Single Source of Truth），管理编辑器的所有状态

    示例

    const state = new EditorState();

    // 监听状态变化
    state.addEventListener('stateChanged', (event) => {
      console.log(`${event.field} changed:`, event.oldValue, '→', event.newValue);
    });

    // 修改状态
    state.setSelectedMaterialIndex(0);
    state.setCurrentModel(modelObject);
