    pickSurface

    • pickSurface(
          e: MouseEvent | TouchEvent,
          target?: Object3D<Object3DEventMap> | Object3D<Object3DEventMap>[],
          recursive?: boolean,
      ): PickResult | null

      拾取模型表面点（包含世界坐标和世界法线） 用于创建热点时获取位置和法线信息

      参数

      • e: MouseEvent | TouchEvent

        鼠标事件或触摸事件

      • 可选target: Object3D<Object3DEventMap> | Object3D<Object3DEventMap>[]

        拾取范围，默认为整个场景

      • recursive: boolean = true

        是否递归检测子对象，默认为true

      返回 PickResult | null

      拾取结果，包含 point（世界坐标）和 normal（世界法线）

      示例

      const result = viewer.picking.pickSurface(event);
      if (result) {
          viewer.hotspotManager.addHotspot({
              position: result.point.toArray(),
              normal: result.normal?.toArray()
          });
      }
