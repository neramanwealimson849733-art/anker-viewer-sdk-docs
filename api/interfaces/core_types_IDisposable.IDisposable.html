    接口 IDisposable

    资源释放接口

    所有持有需要手动释放资源的类（如 Three.js 对象、DOM 元素、事件监听等） 都应该实现此接口，确保资源能够被正确清理，避免内存泄漏。

    示例

    class MyComponent implements IDisposable {
      private _isDisposed = false;
      
      get isDisposed() { return this._isDisposed; }
      
      dispose(): void {
        if (this._isDisposed) {
          console.warn('[MyComponent] Already disposed');
          return;
        }
        
        // 清理资源
        this.geometry?.dispose();
        this.material?.dispose();
        this.removeEventListeners();
        
        this._isDisposed = true;
      }
    }
    interface IDisposable {
        isDisposed: boolean;
        dispose(): void;
    }

    方法

    dispose

    • dispose(): void

      释放所有持有的资源

      调用此方法后，对象将不可再使用。 重复调用应该是安全的（幂等性）。

      应该释放的资源包括：

      • Three.js 对象（Geometry, Material, Texture, RenderTarget 等）
      • DOM 元素引用
      • 事件监听器
      • 定时器和动画帧
      • 其他持有的 IDisposable 对象

      返回 void
