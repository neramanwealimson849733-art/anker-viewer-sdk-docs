    接口 ISceneEditorData

    场景编辑器数据

    用于场景编辑模式，包含：

    • 场景基础信息
    • 场景对象列表（引用模型）
    • 场景全局显示效果（覆盖所有对象的默认效果）
    • 场景相机配置

    重要概念：

    • 场景对象通过 modelId + skuId 引用模型和SKU
    • 场景全局效果会覆盖模型和SKU的默认效果
    • 对象只记录变换、层级、状态，不直接存储材质
