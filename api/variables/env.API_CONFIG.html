    类型声明

    • baseURL: Record<AppEnv, string>

      不同环境的 API 基础 URL

    • useRelativePath: boolean

      是否使用相对路径

      • true: 使用相对路径 "/api"，自动适配当前域名
      • false: 使用 baseURL 中对应环境的绝对路径
    • getBaseURL: function
      • getBaseURL(): string

        获取当前环境的 API 基础 URL

        返回 string
